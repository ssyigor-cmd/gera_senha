# 🏥 Gerador de Senhas - Farmácia Base Regional

Sistema simples para gerenciamento de fila de atendimento em farmácias.

## Funcionalidades
- Geração de senhas Normal e Prioritária
- Chamada do próximo paciente (com prioridade automática)
- Visualização da fila de espera
- Estatísticas de atendimento
- Persistência de dados (salva automaticamente em JSON)

## Como usar
```bash
python main.py
